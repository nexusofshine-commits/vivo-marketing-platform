# vivo营销平台

vivo营销平台是一款移动端广告投放管理工具，提供多账号管理、投放计划管理、智能诊断、数据报表等功能。

## 项目结构

```
├── index.html              # H5版本主页面
├── docs/
│   └── PRD.md             # 产品需求文档
├── wechat-miniprogram/     # 微信小程序版本
│   ├── app.js             # 小程序入口文件
│   ├── app.json           # 小程序配置文件
│   ├── app.wxss           # 小程序全局样式
│   ├── project.config.json # 微信开发者工具配置
│   ├── pages/             # 页面目录
│   │   ├── account-select/ # 账号选择页
│   │   ├── home/          # 首页
│   │   ├── account-manage/ # 账户管理/推广管理页
│   │   ├── diagnosis/      # 智能诊断页
│   │   ├── report/        # 数据报表页
│   │   ├── plan-detail/    # 计划详情页
│   │   └── profile/       # 个人中心页
│   └── utils/
│       └── chart.js       # 图表工具函数
└── README.md              # 项目说明文档
```

## 功能模块

### H5版本
- **账号选择**：多账号分组展示，支持账号状态标识
- **首页**：数据概览、数据趋势图表
- **账户管理/推广管理**：根据账号类型动态切换内容
- **智能诊断**：预警信息、系统通知、详情展开
- **数据报表**：多维度数据展示、时间筛选、图表可视化

### 微信小程序版本
- 与H5版本功能一致
- 支持4个Tab页面导航
- 内置Canvas图表绘制

## 技术栈

### H5版本
- HTML5 + CSS3 + JavaScript
- Canvas API 绘制图表
- 响应式设计，支持移动端

### 微信小程序版本
- 微信小程序原生开发
- Canvas 2D 图表绘制
- 小程序生命周期管理

## 快速开始

### H5版本

直接在浏览器中打开 `index.html` 文件即可预览：

```bash
# 启动本地服务器
python3 -m http.server 8080

# 访问地址
http://localhost:8080/index.html
```

### 微信小程序版本

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `wechat-miniprogram/` 目录
4. 点击"导入"即可预览

## 核心功能说明

### 1. 账号类型适配
- 投放账号登录：显示"推广管理"，内容为计划列表
- 经营主体/多账户管理登录：显示"账户管理"，内容为子账户列表

### 2. 智能诊断
- 预警信息：CTR下降预警、消耗异常等
- 系统通知：新功能上线、安全提醒等
- 支持详情展开/收起

### 3. 数据报表
- 时间筛选：今日/近7日/近30日/自定义
- 多维度切换：概览/计划/账户/创意
- 图表展示：消耗趋势、转化效果

## 数据说明

项目使用虚拟数据进行展示，包含：
- 7天的花费、曝光、点击、CTR数据
- 激活数、次留数等转化数据
- 多个投放计划示例数据
